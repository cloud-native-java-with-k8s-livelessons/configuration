# Basic Configuration 

This example introduces a few key concepts:

 * the `Environment` abstraction 
 * `PropertySource`
 * `@ConfigurationProperties`
 * the refresh Actuator endpoint: `curl localhost:8080/actuator/refresh -d {} -H "Content-Type: application/json" `
 * `@RefreshScope`
 * Spring Cloud Bus
